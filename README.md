# VoxFlow — Releases

Public downloads and the auto-update feed for **VoxFlow**, on-device dictation for macOS.
(The application source lives in a separate private repository.)

## Download

Grab the latest build from the [**Releases**](https://github.com/KaranvirKals/VoxFlow-releases/releases/latest)
page — download `VoxFlow-<version>.dmg`, open it, and drag VoxFlow to Applications.

**Requirements:** macOS 26 or later, Apple Silicon.

## Updates

Once installed, VoxFlow updates itself automatically via [Sparkle](https://sparkle-project.org),
using the `appcast.xml` feed in this repo. The `.zip` on each release is the update payload —
you don't need to download it manually.
